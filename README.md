# React Pipeline    

Initialize the Project   
```bash
# Create Vite project
yarn create vite your-styles --template react-ts
cd your-styles

# Install dependencies
yarn add redux react-redux @reduxjs/toolkit

# Install SCSS support
yarn add sass

# Install Jest for testing
yarn add --dev jest @testing-library/react @testing-library/jest-dom @testing-library/user-event vitest jsdom

# Install Storybook
npm create storybook@latest  # select for testing not documentation   
```

Run locally   
```bash
# react
$ yarn dev

# test
$ yarn test

# build project
$ yarn build

# storybook
$ yarn storybook   
```
